### <u>HPE OneView 5.00 Library</u>

## Set-HPOVRemoteSupportPrimaryContact
<p>
Set HPE OneView Remote Support default primary contact.

### SYNTAX
<p>
<pre><code>Set-HPOVRemoteSupportPrimaryContact [-InputObject] &lt;Object&gt;[ [-ApplianceConnection] &lt;Object&gt;][ [-WhatIf] &lt;SwitchParameter&gt;][ [-Confirm] &lt;SwitchParameter&gt;] [&lt;CommonParameters&gt;]</code></pre>

### Detailed Description
<p>
Register with Hewlett Packard Enterprise to allow automatic case creation for hardware failures on servers and enclosures and to enable Proactive Care. Once enabled, all eligible devices added in the future will be automatically enabled for remote support.

Use this Cmdlet to change or update the HPE OneView Remote Support primary Contact.

Required permissions: Infrastructure administrator
