## Set-HPOVApplianceProxy
<p>
Configure appliance web proxy.

### SYNTAX
<p>
<pre><code>Set-HPOVApplianceProxy [-Hostname] &lt;String&gt; [-Port] &lt;Int&gt;[ [-Https] &lt;SwitchParameter&gt;][ [-Async] &lt;SwitchParameter&gt;][ [-ApplianceConnection] &lt;Object&gt;] [&lt;CommonParameters&gt;]</code></pre>
 <pre><code>Set-HPOVApplianceProxy [-Hostname] &lt;String&gt; [-Port] &lt;Int&gt; [-Username] &lt;String&gt; [-Password] &lt;SecureString&gt;[ [-Https] &lt;SwitchParameter&gt;][ [-Async] &lt;SwitchParameter&gt;][ [-ApplianceConnection] &lt;Object&gt;] [&lt;CommonParameters&gt;]</code></pre>

### Detailed Description
<p>
