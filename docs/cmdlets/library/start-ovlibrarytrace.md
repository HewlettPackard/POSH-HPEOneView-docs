# Start-OVLibraryTrace

## Syntax

```powershell
Start-OVLibraryTrace
    [-Location <String>]

```

## Description

This Cmdlet is deprecated.  Please use Get-OVCommandTrace.

## Examples

###  Example 1 

```powershell

```
