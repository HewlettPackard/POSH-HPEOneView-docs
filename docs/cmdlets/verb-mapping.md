# Verb mapping

The following table shows the available HPE OneView PowerShell Verbs and which map to the HPE OneView actions.

| HPE OneView Actions Verb | HPE OneView PowerShell Verb |
| :--- | :--- |
| Add | Add |
| Create | New |
| Edit | Set |
| Remove | Delete |
| Refresh | Update |
| Reapply | Update |
| PowerOn | Start |
| PowerOff | Stop |
