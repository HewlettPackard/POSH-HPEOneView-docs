### <u>HPE OneView 5.00 Library</u>

## Enable-HPOVLdapLocalLogin
<p>
Enable local authentication logins.

### SYNTAX
<p>
<pre><code>Enable-HPOVLdapLocalLogin [-ApplianceConnection] &lt;Object&gt;[ [-WhatIf] &lt;SwitchParameter&gt;][ [-Confirm] &lt;SwitchParameter&gt;] [&lt;CommonParameters&gt;]</code></pre>

### Detailed Description
<p>
Use this cmdlet to set enable appliance local logins.  An existing authentication directory must be configured in order to use this CMDLET.


### Parameters
